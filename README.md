# Deep Learning for Natural Language Processing

## Overview

This repository contains the assignments, practical projects, and study materials developed during the **Deep Learning for Natural Language Processing** graduate-level course.

The course explored fundamental and advanced Natural Language Processing (NLP) concepts, combining traditional language processing techniques with modern Deep Learning approaches.

Throughout the course, practical assignments were implemented using Python and Jupyter Notebooks to investigate word embeddings, part-of-speech tagging, sentiment analysis, and other NLP tasks.

## Technologies

- Jupyter Notebook
- TensorFlow
- Keras

- Natural Language Processing (NLP)

## Repository Contents

### Practical Assignment 1 – Word2Vec

Implementation and analysis of **Word2Vec** embeddings.

Topics covered include:

- Word embeddings
- Vector representations
- Semantic similarity
- Continuous Bag of Words (CBOW)
- Skip-Gram

---

### Practical Assignment 2 – Part-of-Speech Tagging

Implementation of a **Part-of-Speech (POS) Tagger**.

Topics include:

- Sequence labeling
- POS classification
- Natural Language Processing techniques
- Model evaluation

---

### Final Project – Twitter Sentiment Analysis

Development of a sentiment analysis model capable of classifying tweets according to their sentiment.

Topics include:

- Text preprocessing
- Tokenization
- Feature extraction
- Deep Learning for text classification
- Sentiment analysis

---
